Welcome!

This website can be used to generate stl files for many of the items available in the game Destiny. The files are created using the same geometry components as the mobile and web previewers available from the Bungie website and mobile app. The source code for this website can be found on the GitHub repository.

Usage

  1. Enter an item name in the text field above.
  2. Copy the generated text to your clipboard.
  3. Past the contents into a text editor and save with the stl file extension.

Notes

  1. This is a beta version and not all items have been tested.
  2. Many of the generated models will require a quick clean up of 2D surfaces and holes before they can be 3D printed.
  3. Please submit any issues to the GitHub repository.
