Usage:

  1. Enter an item name or id in the text field above.
  2. Copy the generated text to your clipboard.
  3. Past the contents into a plaintext editor and save the file with the .stl extension.
  4. Open the file in your favorite CAD program or a 3D printing utility such as Netfabb.

Notes:

  1. This is a beta version of the tool and not all items have been tested.
  2. Many of the generated models will require a quick clean up of 2D surfaces and holes before they can be 3D printed.
  3. Please submit any issues to the GitHub repository.
