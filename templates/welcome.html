Welcome!

This website can be used to generate stl files for many of the items available in the game Destiny. The files are created with the same geometry components used in the gear previewer available on the Bungie website and mobile app. The source code for this website can be found on the GitHub repository.



Usage

  1. Enter an item name in the text field above such as an armour, weapon or ship.
  2. Copy the generated text to your clipboard.
  3. Past the contents into a text editor and save with the stl file extension.
  4. Import the stl file into your favorite CAD program or a 3D printing utility such as NetFabb.

Notes

  1. This is a beta version and not all items have been tested.
  2. Many of the generated models will require a quick clean up of 2D surfaces and holes before they can be 3D printed.
  3. Please submit any issues to the GitHub repository.
