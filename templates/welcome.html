Welcome!

This website can be used to generate stl files for thousands of in game items found in Destiny. The files are created with the same source geometry used to render your character on the official Destiny mobile app and Bungie website. This project is constantly being updated with new features and is still in early development. For the latest updates you can follow the project on Twitter.



Usage

  1. Select an item and click generate.
  2. Download the generated .stl file.
  3. Import the .stl file into your favorite CAD program or a 3D printing clean up utility such as NetFabb Cloud.

Notes

  1. This is a beta version and not all items have been tested.
  2. It is recommended to clean up any models using a tool such as NetFabb Cloud before they can be 3D printed.
  3. Please report any issues to the project Twitter page.
